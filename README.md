# Employee Performance Analyzer

## Introduction
Employee performance evaluation is essential for organizations to measure productivity and efficiency.

## Problem Statement
Manual performance evaluation is time-consuming and prone to bias.

## Solution
This project builds an automated system that evaluates employee performance using Python.

## Features
- Performance scoring
- Attendance analysis
- Work quality evaluation
- Teamwork and communication analysis

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## How to Run
1. Download the project
2. Run the Python file
3. Enter employee details
4. Get performance analysis
